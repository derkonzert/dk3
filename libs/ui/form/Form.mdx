## TextInput

import { TextInput } from "./TextInput"

<Playground>
  <State initial={{ value: "" }}>
    {({ state, setState }) => (
      <TextInput
        id="test"
        label="Band Name"
        value={state.value}
        onChange={e => setState({ value: e.target.value })}
      />
    )}
  </State>

Invalid text
