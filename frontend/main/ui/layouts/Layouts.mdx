## ListAndDetail

This layout component is not well suited for a component based demonstration.

```jsx
import {
  ListAndDetail,
  ListAndDetailMain,
  ListAndDetailSide,
} from "@dk3/ui/layouts/ListAndDetail"

render(
  <ListAndDetail showDetail={state.showDetail}>
    <ListAndDetailMain>Main page content</ListAndDetailMain>
    <ListAndDetailSide>Detail page content</ListAndDetailSide>
  </ListAndDetail>
)
```
